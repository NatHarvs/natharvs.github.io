Personal Projects

AI Flocking

Using C++ and following the simple rules of cohesion, alignment and separation made by Craig Reynolds, these boids(little squares) were created to behave as a flock. Skeleton code was given that had the squares already made and a few mathematical functions were provided that did arithmetic on 3D vectors. To expand on this project a few obstacles could be added for the agents to avoid and the wall clipping could be perfected. Coded solution: https://github.com/NatHarvs/Flocking-AI https://www.youtube.com/watch?v=igk0LRGxSxg
